# 贪吃蛇游戏

这是一个适合放在 GitHub 上的第一个小项目：纯 HTML、CSS 和 JavaScript 编写，不需要安装依赖，打开网页就能玩。

## 玩法

- 使用方向键或 `WASD` 控制移动
- 按空格暂停或继续
- 吃到食物会得分并逐渐加速
- 撞墙或撞到自己会结束游戏
- 最佳分数会保存在本地浏览器中

## 本地运行

直接打开 `index.html`，或在项目目录中启动任意静态服务器。
